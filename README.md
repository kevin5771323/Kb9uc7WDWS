# 家政服务管理系统 java854

## 项目概述

家政服务管理系统是一个基于前后端分离架构的信息管理系统，旨在提供用户友好的服务预约、管理和分配流程。系统支持管理员、用户和服务人员三种角色，以实现高效的沟通与协作。

## 技术栈

- **前端技术**：Vue, Ajax, Json
- **后端技术**：SpringBoot, MyBatisPlus
- **数据库**：MySql

## 功能模块

### 用户功能

- 首页浏览
- 服务信息查看
- 公告信息浏览
- 留言反馈
- 个人中心管理
- 后台简单管理

### 管理员功能

- 首页总览
- 个人中心管理
- 用户管理
- 服务人员管理
- 服务信息管理
- 服务类型管理
- 服务预约管理
- 服务取消管理
- 服务分配管理
- 服务进度管理
- 评价信息管理
- 留言反馈处理
- 系统全面管理

### 服务人员功能

- 首页快速导航
- 个人中心管理
- 服务分配查看
- 服务进度更新
- 评价信息浏览

## 系统角色

- 用户：使用系统预约服务，查看服务信息等。
- 管理员：负责系统运营，包括用户、服务人员、服务类型及系统设置管理。
- 服务人员：接受服务分配，更新服务进度，接收评价信息。

## 运行环境

- 前端：现代浏览器（建议使用最新版本的Chrome、Firefox等）
- 后端：JDK 1.8及以上版本，MySql数据库
- 服务器：能够运行Java应用服务器环境

## 部署步骤

1. 准备MySql数据库环境
2. 导入项目提供的SQL文件
3. 部署后端SpringBoot应用
4. 部署前端Vue应用
5. 配置前后端交互
